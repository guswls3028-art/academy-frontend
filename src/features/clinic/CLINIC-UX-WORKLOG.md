## 10점을 위한 잔여 작업

### Round 1: 표현 일관성 패스 (모든 탭)
- [ ] 모든 user-facing 텍스트에서 개발자 용어 제거
- [ ] 빈 상태 메시지 통일 (모두 CTA 포함)
- [ ] 버튼 라벨 동사형으로 통일 ("만들기", "확인하기", "선택하기")

### Round 2: 오늘 탭 → 9점
- [ ] 오늘 출석 현황 한 줄 요약 추가 (타임라인 위: "오늘 출석 12/15명")
- [ ] 과거 세션 미니 진행률 → 색상 강화
- [ ] 액션바 → 승인 없을 때 완전 숨김 확인
- [ ] "출석 확인 →" 링크가 해당 세션으로 직접 이동 (date param)

### Round 3: 클리닉 진행 탭 → 9점
- [ ] 세션 선택 탭 스트립 (사이드바 대신 상단 수평 탭)
- [ ] 참여자 카드: 사유별 아이콘 강화
- [ ] 전체 출석 후 완료 상태 표시 ("모든 학생 확인 완료 ✓")
- [ ] 빈 세션: "아직 예약된 학생이 없습니다" + "예약 탭에서 추가 →"

### Round 4: 예약 탭 → 9점
- [ ] 승인 대기 / 대상자 선택 구분 더 명확히 (시각적 분리)
- [ ] 대상자 카드에 session_title 표시 확인
- [ ] 선택 상태 실시간 반영 (toolbar 인원수 + 하단 플로팅 바)
- [ ] "전체 해제" 버튼 추가

### Round 5: 일정 관리 탭 → 9점
- [ ] 세션 카드에 예약 인원 시각적 강화 (프로그레스 바)
- [ ] 상태 dot → 텍스트 라벨로 병기 ("여유" / "거의 참" / "마감")
- [ ] 캘린더 오늘 날짜 강조 확인
- [ ] "클리닉 만들기" 모달 내 날짜 중복 선택 방지

### Round 6: 최종 리뷰
- [ ] 전체 탭 사용자 관점 재평가
- [ ] 표현 검증 (강사가 동료에게 말하는 수준)
- [ ] 빈 상태 / 로딩 상태 / 에러 상태 모두 확인

## 작업 로그

### 2026-03-15 — Round 1~3 완료

**Round 1: 표현 일관성 패스** ✅
- ClinicCreatePanel: "클리닉 생성"→"클리닉 만들기", 성공/실패 메시지 인간화
- ClinicConsoleWorkspace: "재시험 연결"→"재시험 관리", "대상자"→"학생", 타이틀 자연어화
- ClinicOperationsPage: "조회만 가능합니다"→"지난 날짜입니다"
- ClinicBookingsPage: "모든 신청이 처리되었습니다"→"처리 완료"
- ClinicDaySchedulePanel: 빈 상태 문구 단순화

**Round 2: 오늘 탭 → 9점** ✅
- 출석 현황 한 줄 요약 추가 ("오늘 전체 N명 중 출석 M명")
- "출석 확인→" 클릭 시 date param 전달 (자동 날짜 로드)
- 액션바 0건일 때 완전 숨김
- 미예약 배너: 대상자 없으면 숨김
- 장소 텍스트 truncation 방지

**Round 3: 클리닉 진행 + 예약 → 9점** ✅
- 클리닉 진행: 전원 확인 완료 시 "모든 학생 확인 완료 ✓", 빈 세션 CTA, 출석/불참 카드 시각 피드백
- 예약: 승인/대상 구분선 추가, 플로팅 선택 바, "전체 해제" 버튼, 빈 대상 안내 강화

**Round 4: 일정 관리 → 9점** ✅
- 세션 카드에 용량 프로그레스 바 추가 (초록/노랑/빨강)
- 상태 라벨 텍스트 표시 ("여유"/"거의 참"/"마감") — hover 불필요
- 과거 날짜 세션 카드 muted + 삭제 버튼 숨김
- 캘린더 오늘 하이라이트 확인 (기존 작동)
- 생성 모달 날짜 동기화 확인 (기존 작동)

**Round 5: 최종 리뷰 + 잔여 수정** ✅
- 리뷰 결과: 전체 8.8/10 (오늘 9.0, 일정 8.8, 예약 9.2, 클리닉 진행 8.6)
- 수정 1: 클리닉 진행 — 메모 중복 표시 제거 (reason-detail fallback 삭제)
- 수정 2: 일정 관리 — dead code 제거 (useClinicParticipants, selectedSessionId, mode, rangeFromMode 등)
- 수정 후 예상: 전체 9.0+
