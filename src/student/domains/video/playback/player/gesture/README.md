# 학생 앱 비디오 제스처 (YouTube 스타일)

## 개요

- **더블탭**: 왼쪽 절반 → -10초, 오른쪽 절반 → +10초
- **탭 스태킹**: 600ms 이내 추가 탭마다 ±10초 누적 (10 → 20 → 30…)
- **오버레이**: 중앙에 "» 20s" / "« 20s" + 리플 애니메이션 (토스트 대체)
- **이벤트**: Pointer 이벤트만 사용 (마우스·터치 통합)

## 타이밍

| 상수 | 값 | 설명 |
|------|-----|------|
| DOUBLE_TAP_MS | 300ms | 두 번째 탭으로 인정하는 시간 |
| STACK_WINDOW_MS | 600ms | 연속 탭 시 누적 구간 (탭 시마다 리셋) |
| OVERLAY_FADE_AFTER_MS | 800ms | 오버레이 표시 후 페이드 아웃 |
| COOLDOWN_MS | 400ms | 오버레이 종료 후 다음 제스처 허용 대기 |
| DRAG_THRESHOLD_PX | 10px | 이 이상 이동 시 탭이 아닌 드래그로 처리 |

## 존 (50% 분할)

- **0**: 왼쪽 절반 → -10초
- **1**: 가운데 → 재생/일시정지 토글 (싱글탭 시)
- **2**: 오른쪽 절반 → +10초

## 상태 머신

- **IDLE** → 첫 탭 시 **WAITING_FOR_SECOND_TAP** (300ms 타이머)
- **WAITING_FOR_SECOND_TAP**에서 두 번째 탭:
  - 좌/우면 seek + **STACKING** (600ms 타이머, 탭 시 리셋)
  - 가운데/탭 없음이면 싱글탭 처리 후 **IDLE**
- **STACKING**에서 600ms 무탭 → **ANIMATING** → 오버레이 800ms 후 **COOLDOWN** → **IDLE**

## 무시 대상

- `.svpControls`, `.svpBigPlay`, `.svpProgressRow`, `.svpRange` 내부 포인터
- 볼륨 스와이프 등으로 설정된 `getIsDrag() === true` 인 경우
