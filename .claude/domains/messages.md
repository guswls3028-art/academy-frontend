## 메시지 발송 모달 (SendMessageModal.tsx)

### 구조
- 좌: 수신자 카드 + 실시간 미리보기 (SMS: iPhone 버블, 알림톡: 카카오 카드)
- 우: SMS 직접 작성 또는 알림톡 템플릿 선택
- 기본 모드: SMS (smsAllowed일 때), 없으면 알림톡 자동 전환
- 탭 순서: SMS → 알림톡

### SMS 모드
- 본문 직접 작성 가능
- 템플릿 드롭다운: 최근 사용 / 기본 템플릿 / 내 템플릿 + 검색
- 변수 블록 커서 삽입 (도메인별 필터링)
- 글자수 표시 (SMS 90자 / LMS 2000자)
- 템플릿 저장 기능

### 알림톡 모드
- 본문 readOnly (편집 불가)
- 승인된 템플릿 선택 전용
- #{내용} 변수 포함 시 자유 입력 영역 제공
- 변수 상태 표시 (자동/제공/미제공)

### 공통
- 도메인 배지 (학생/출결/성적/강의/시험/직원)
- 구체적 발송 버튼: "학부모 3명 + 학생 3명에게 문자 발송"
- 발송 불가 사유 표시

## 성적 발송 양식 시스템

### 동작 원리
1. "수업결과 발송" 클릭 시 `grades` 카테고리 템플릿을 API에서 조회
2. 커스텀 템플릿(이름이 `[학원플러스]`로 시작하지 않는 것)이 있으면 자동 로드
3. `substituteScoreVars()` 함수가 템플릿 변수를 실제 성적 데이터로 치환
4. 치환된 결과가 SMS 본문에 자동 입력 (사용자 편집 가능)
5. 없으면 기존 `generateScoreReport()` 기본 양식 사용

### 지원 변수 (substituteScoreVars)
| 변수 | 설명 | 예시 값 |
|------|------|---------|
| `#{학생이름}` | 학생 이름 | 홍길동 |
| `#{강의명}` | 강의 제목 | 수학 심화반 |
| `#{차시명}` | 차시 제목 | 3차시 (18:00~21:00) |
| `#{날짜}` | 오늘 날짜 | 4월 1일 |
| `#{시험1}` ~ `#{시험10}` | N번째 시험 점수 | 25 (없으면 "-") |
| `#{시험1만점}` ~ `#{시험10만점}` | N번째 시험 만점 | 30 (없으면 "-") |
| `#{시험1명}` ~ `#{시험10명}` | N번째 시험 이름 | 어휘 |
| `#{과제1}` ~ `#{과제10}` | N번째 과제 점수 | 85 |
| `#{과제1만점}` ~ `#{과제10만점}` | N번째 과제 만점 | 100 |
| `#{시험총점}` | 시험 점수 합계 | 285 |
| `#{시험총만점}` | 시험 만점 합계 | 350 |
| `#{숙제완성도}` | 과제 완성 요약 | 3/3 완료 |
| `#{시험성적}` | 성적 상세 블록 (buildScoreDetail) | [시험]\n- 어휘: 25/30... |

### 양식 관리
- 메시지 > 템플릿 관리에서 생성/수정/삭제 가능
- 카테고리를 "성적"으로 설정해야 수업결과 발송 시 자동 로드됨
- 이름이 `[학원플러스]`로 시작하지 않아야 커스텀 양식으로 인식
- 테넌트별 독립 — 각 학원이 자체 양식 관리

## 뿌리오 연동

### 인증 구조
- HTTPBasicAuth(accountID, apiKey) → Bearer Token (24시간)
- Token 발급 시 IP 화이트리스트 체크됨

### 필수 설정
1. 뿌리오 계정 ID + API 인증키 (메시지 > 설정에서 입력)
2. 발신번호 (뿌리오에 등록된 번호)
3. 연동 IP 등록 (뿌리오 관리자 > 연동관리 > 연동IP)
   - 메시징 워커 EIP: 43.201.119.172

### 주의사항
- "연동 테스트" 버튼은 API 서버에서 실행 → 워커 IP와 다르면 토큰 발급 실패
- 실제 발송은 워커를 통해 실행되므로 워커 IP만 등록되면 발송 정상 동작
- 워커 EIP 분리 주의: ASG 교체 시 EIP 재연결 필요

## E2E 테스트

### 실행 플로우 (Tenant 1 기준)
1. 강의 > 차시 > 성적 탭 이동
2. "대상자 전원등록" 클릭 (수강생 세션 배정)
3. "시험 추가" > "신규 시험" > 시험명/만점 입력 > "일괄 생성"
4. 새로고침 → 학생 목록 + 시험 컬럼 표시
5. 학생 체크박스 선택
6. "수업결과 발송" 클릭
7. 모달에서 양식 자동 로드 확인
8. 발송

### 주의: 세션 수강생 배정
