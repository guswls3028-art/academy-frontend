# 학생 사용 가이드

## 시작하기

### 처음 가입하면
- 선생님이 학원에 등록하면 **카카오톡 알림톡**으로 아이디와 비밀번호가 전송됩니다
- 학부모님 전화번호로도 같은 정보가 전송됩니다
- 알림톡에 있는 **접속 링크**를 눌러 로그인하세요

### 직접 가입하기
1. 학원 앱 주소에 접속합니다
2. **회원가입** 버튼을 누릅니다
3. 이름, 전화번호, 비밀번호를 입력합니다
4. 선생님이 승인하면 로그인할 수 있습니다

### 로그인
1. 학원 앱 주소에 접속합니다
2. **로그인** 버튼을 누릅니다
3. 아이디와 비밀번호를 입력합니다
4. **로그인**을 누르면 홈 화면이 나타납니다

---

## 홈 화면

로그인하면 가장 먼저 보이는 화면입니다.

**화면 구성:**
- **공지사항**: 학원 공지가 위쪽에 표시됩니다. 터치하면 자세히 볼 수 있습니다
- **오늘 할 일**: 미완료 과제, 재시험, 클리닉 예약 등이 표시됩니다
- **빠른 메뉴**: 영상, 시험, 제출, 성적, 보관함, 클리닉으로 바로 이동

**하단 탭 5개:**
| 탭 | 설명 |
|------|------|
| 🏠 홈 | 대시보드 (지금 보고 있는 화면) |
| ▶️ 영상 | 수업 영상 시청 |
| 📅 일정 | 수업 일정 달력 |
| 🔔 알림 | 새 소식 확인 |
| 💬 커뮤니티 | 질문, 공지, 자료 |

---

## 영상 보기

**하단 탭 → 영상**

### 영상 찾기
1. 수강 중인 강의가 카드 형태로 나타납니다
2. 강의 카드를 터치합니다
3. 차시(수업) 목록이 표시됩니다
4. 보고 싶은 차시를 터치합니다
5. 영상 목록에서 영상을 선택하면 재생이 시작됩니다

### 영상 시청
- 재생/일시정지: 화면 터치
- 이어보기: 지난번 시청 위치부터 자동 재생
- 좋아요: ❤️ 버튼
- 댓글: 영상 아래에서 댓글을 남길 수 있습니다

---

## 일정 확인

**하단 탭 → 일정**

### 달력 보기
- 수업이 있는 날짜에 **초록색 점**이 표시됩니다
- 날짜를 터치하면 그 날의 수업 목록이 아래에 나타납니다

### 수업 상세
- 수업 항목을 터치하면 상세 정보가 표시됩니다:
  - **영상 보기**: 해당 수업의 영상으로 이동
  - **시험 보기**: 해당 수업의 시험으로 이동
  - **과제 제출**: 과제 제출 페이지로 이동
  - **성적 보기**: 성적 확인 페이지로 이동

---

## 시험 보기

**홈 → 시험** 또는 **더보기 → 시험**

### 시험 목록
- 응시 가능한 시험이 목록으로 표시됩니다
- **빨간색**: 마감이 임박한 시험 (24시간 이내)
- **파란색**: 여유 있는 시험
- **회색**: 아직 열리지 않았거나 마감된 시험

### 시험 응시
1. 시험을 터치합니다
2. **시험 보기** 버튼을 누릅니다
3. 문항별로 답을 입력합니다
   - 답을 입력하면 자동 저장됩니다 (중간에 나가도 복원)
4. 모든 문항을 풀었으면 **제출하기**를 누릅니다
5. 확인 팝업에서 **제출**을 누르면 완료됩니다

### 시험 결과 확인
- 제출 후 자동으로 결과가 표시됩니다
- 점수, 합격/불합격 여부를 확인합니다
- 선생님이 정답 공개를 설정했으면 정답도 볼 수 있습니다
- 재시험이 가능하면 **재시험 보기** 버튼이 나타납니다

---

## 성적 확인

**홈 → 성적** 또는 **더보기 → 성적**

### 시험 성적
- 응시한 모든 시험의 점수와 합격 여부가 표시됩니다
- 시험을 터치하면 문항별 결과를 볼 수 있습니다

### 과제 성적
- 아래쪽에 과제별 점수가 표시됩니다

---

## 과제/성적표 제출

**홈 → 제출** 또는 **더보기 → 제출**

### 성적표 제출
1. **성적표 제출**을 선택합니다
2. 사진을 찍거나 파일을 선택합니다
3. **제출** 버튼을 누릅니다

### 과제 제출
1. **과제 제출**을 선택합니다
2. 사진이나 영상을 선택합니다
3. **제출** 버튼을 누릅니다

---

## 클리닉 (보충 수업)

**홈 → 클리닉** 또는 **더보기 → 클리닉**

### 예약하기
1. **예약** 탭에서 날짜를 선택합니다
2. 가능한 시간대가 표시됩니다
   - **초록**: 자리 있음
   - **노랑**: 거의 마감
   - **빨강**: 마감
3. 원하는 시간을 터치합니다
4. 메모를 입력할 수 있습니다 (선택)
5. **예약** 버튼을 누릅니다

### 내 예약 확인
- **내 일정** 탭에서 예약한 클리닉을 확인합니다
- 대기 중(선생님 승인 전) / 확정됨 상태가 표시됩니다
- 대기 중인 예약은 **취소** 버튼으로 취소할 수 있습니다

### 클리닉 인증 패스
- **더보기 → 클리닉 인증 패스**
- 클리닉 교실 입장 시 이 화면을 선생님에게 보여주세요
- 실시간 시계와 배경색으로 본인 확인이 됩니다
- 매초 색상이 바뀌어서 스크린샷으로는 사용할 수 없습니다

---

## 커뮤니티

**하단 탭 → 커뮤니티**

### 공지사항
- **공지사항** 탭에서 학원 공지를 확인합니다
- 터치하면 전체 내용을 볼 수 있습니다

### 게시판
- **게시판** 탭에서 자유 게시판 글을 봅니다

### 자료실
- **자료실** 탭에서 학습 자료를 다운로드합니다

### 질문하기 (QnA)
1. **QnA** 탭으로 이동합니다
2. **질문하기** 버튼을 누릅니다
3. 제목과 내용을 입력합니다
4. **질문 보내기**를 누릅니다
5. 선생님이 답변하면 알림이 옵니다
6. 다시 QnA 탭에서 내 질문을 터치하면 답변을 확인할 수 있습니다

### 상담 신청
1. **상담 신청** 탭으로 이동합니다
2. 제목과 내용을 입력합니다
3. **보내기**를 누릅니다
4. 선생님이 답변하면 확인할 수 있습니다

---

## 알림

**하단 탭 → 알림**

- 최근 7일간의 알림이 표시됩니다:
  - 클리닉 예약 승인
  - QnA 답변 도착
  - 새 공지사항
- 알림을 터치하면 해당 페이지로 이동합니다

---

## 보관함 (내 파일)

**홈 → 보관함** 또는 **더보기 → 내 인벤토리**

- 제출한 파일과 선생님이 공유한 자료가 보관됩니다
- **새 폴더**: 폴더를 만들어 정리할 수 있습니다
- **업로드**: 파일을 추가할 수 있습니다 (최대 100MB)
- 파일을 터치하면 다운로드됩니다
- 길게 누르면 삭제할 수 있습니다

---

## 프로필

**더보기 → 프로필**

### 내 정보 수정
- **편집** 버튼을 눌러 이름, 전화번호, 학교 정보를 수정합니다
- 프로필 사진을 변경할 수 있습니다 (최대 5MB)

### 아이디 변경
- **아이디 변경하기**를 터치합니다
- 새 아이디를 입력하고 저장합니다

### 비밀번호 변경
- **비밀번호 변경하기**를 터치합니다
- 현재 비밀번호와 새 비밀번호를 입력합니다
- **변경** 버튼을 누릅니다

**더보기 → 출결 현황**

- 출석 정보는 각 수업의 차시 상세에서 확인할 수 있습니다
- **수업 목록 보기**를 터치하면 일정 페이지로 이동합니다

**더보기 → 로그아웃**

- 빨간색 **로그아웃** 버튼을 터치합니다
- 로그인 화면으로 돌아갑니다

## 자주 묻는 질문

### 비밀번호를 잊었어요
- 로그인 화면에서 **비밀번호 찾기**를 누릅니다
- 등록된 전화번호로 인증번호가 발송됩니다
- 인증 후 임시 비밀번호가 카카오톡으로 전송됩니다

### 시험 답을 실수로 제출했어요
- 선생님이 재시험을 허용했으면 **재시험 보기** 버튼이 나타납니다
- 허용되지 않았으면 선생님에게 문의하세요

### 영상이 안 재생돼요
- 인터넷 연결을 확인하세요
- 앱을 새로고침(위에서 아래로 당기기)해 보세요
- 계속 안 되면 선생님에게 문의하세요

### 클리닉 예약을 취소하고 싶어요
- 아직 승인 전(대기 중)이면 **내 일정**에서 취소할 수 있습니다
- 이미 승인된 예약은 선생님에게 취소를 요청하세요

### 화면이 이상하게 보여요
- 새로고침을 해 보세요
- "업데이트가 있습니다" 메시지가 나오면 **새로고침** 버튼을 누르세요
