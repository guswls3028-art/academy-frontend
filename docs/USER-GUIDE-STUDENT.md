# 학생 사용 가이드

---

## 1. 시작하기

### 1-1. 가입하기

**선생님이 등록한 경우:**
- 카카오톡으로 알림톡이 옵니다
- 알림톡에 **아이디**, **비밀번호**, **접속 링크**가 적혀 있습니다
- 링크를 눌러서 바로 로그인할 수 있습니다

**직접 가입하는 경우:**
1. 학원 앱 주소에 접속합니다
2. **회원가입** 버튼을 누릅니다
3. 이름, 전화번호, 비밀번호를 입력합니다
4. 선생님이 승인하면 로그인할 수 있습니다
5. 승인되면 카카오톡으로 알림이 옵니다

> 💡 학부모 계정도 함께 만들어집니다. 학부모 아이디는 **학부모 전화번호**, 비밀번호는 **0000**입니다.

### 1-2. 로그인
1. 학원 앱 주소에 접속합니다
2. **로그인** 버튼을 누릅니다
3. 아이디와 비밀번호를 입력합니다
4. **로그인**을 누르면 홈 화면이 나타납니다

### 1-3. 비밀번호를 잊었을 때
1. 로그인 화면에서 **비밀번호 찾기**를 누릅니다
2. 등록된 전화번호를 입력합니다
3. 카카오톡으로 **인증번호**가 옵니다 (10분 이내 입력)
4. 인증 후 **임시 비밀번호**가 카카오톡으로 전송됩니다
5. 임시 비밀번호로 로그인한 후 반드시 비밀번호를 변경하세요

## 2. 홈 화면

로그인하면 가장 먼저 보이는 화면입니다.

### 화면 구성

**위쪽:**
- 📢 **공지사항** — 학원 공지 미리보기 (터치하면 전체 보기)

**가운데:**
- ⚡ **오늘 할 일** (해당되는 경우만 표시)
  - 불합격한 과제가 있으면 "과제 미완료 N건" 표시
  - 재시험이 필요하면 "재시험 N건" 표시
  - 클리닉 예약이 있으면 표시

**아래쪽: 빠른 메뉴 (6개 타일)**

| 메뉴 | 설명 |
|------|------|
| ▶ 영상 | 수업 영상 시청하기 |
| 📝 시험 | 시험 목록 보기 / 응시하기 |
| 📤 제출 | 성적표나 과제 제출하기 |
| ⭐ 성적 | 내 성적 확인하기 |
| 📦 보관함 | 내 파일 관리 |
| 🏥 클리닉 | 보충 수업 예약하기 |

**하단 탭바 (항상 표시):**

| 🏠 홈 | ▶ 영상 | 📅 일정 | 🔔 알림 | 💬 커뮤니티 |
|--------|--------|--------|--------|------------|

## 3. 영상 보기

### 3-1. 영상 찾기

**하단 탭 → 영상**

1. 수강 중인 강의가 카드로 표시됩니다
   - 각 카드: 강의명, 영상 수, 총 재생 시간
2. 강의 카드를 터치합니다
3. 차시(수업) 목록이 나타납니다
   - 각 차시: 제목, 영상 수, 시간
4. 차시를 터치하면 영상 목록이 나옵니다
5. 영상을 터치하면 재생이 시작됩니다

### 3-2. 영상 시청

- **재생/일시정지**: 화면을 터치합니다
- **이어보기**: 지난번에 본 위치에서 자동으로 이어서 재생됩니다
- **좋아요**: ❤️ 버튼으로 좋아요를 누를 수 있습니다
- **댓글**: 영상 아래에서 댓글을 남길 수 있습니다
- **다음 영상**: 영상이 끝나면 다음 영상으로 자동 전환 카운트다운이 나옵니다

> 💡 영상 시청 기록은 자동으로 저장됩니다. 다시 들어오면 멈춘 위치부터 재생됩니다.

## 4. 일정 확인

### 4-1. 달력

**하단 탭 → 일정**

- 수업이 있는 날짜에 **색상 점**이 표시됩니다
  - 🟢 초록: 예정된 수업
  - 🟡 노랑: 진행 중인 수업
  - ⚫ 회색: 지난 수업
- 날짜를 터치하면 그 날의 수업 목록이 아래에 나타납니다

### 4-2. 수업 상세

수업 항목을 터치하면 상세 페이지가 열립니다.

**이동할 수 있는 곳:**
- **영상 보기**: "이 수업의 영상을 볼 수 있어요" → 해당 차시 영상 목록으로 이동
- **시험 보기**: 해당 차시의 시험으로 이동 (시험이 있는 경우)
  - 시험이 1개면 "시험 보기"
  - 시험이 여러 개면 "1차 시험", "2차 시험" 등으로 표시
- **과제 제출**: 과제 제출 페이지로 이동
- **성적 보기**: 성적 확인 페이지로 이동

## 5. 시험

### 5-1. 시험 목록

**홈 → 시험** 또는 **더보기 → 시험**

| 표시 색상 | 의미 |
|----------|------|
| 🔴 빨간색 | 마감이 24시간 이내! 서둘러 응시하세요 |
| 🔵 파란색 | 응시 가능 (여유 있음) |
| ⚫ 회색 | 아직 열리지 않았거나 이미 마감 |

### 5-2. 시험 응시

1. 시험을 터치합니다
2. **시험 보기** 버튼을 누릅니다
3. 문항별로 답을 입력합니다
   - 객관식: 1~5번 중 선택
   - 주관식: 답을 직접 입력
4. **제출하기**를 누릅니다
5. 확인 팝업:
   - 안 푼 문항이 있으면 "X개 문항 미응답" 안내
   - **제출**을 누르면 완료

> 💡 **중간 저장**: 답을 입력하면 자동으로 저장됩니다. 실수로 나갔다가 다시 들어와도 이전 답이 그대로 있습니다.

> ⚠️ 한 번 제출하면 수정할 수 없습니다. 제출 전에 꼭 확인하세요!

### 5-3. 시험 결과

제출 후 자동으로 결과가 표시됩니다:

- **점수**: 받은 점수 / 총점 (예: 85 / 100)
- **합격 여부**: 합격 ✅ 또는 불합격 ❌
- **문항별 결과** (선생님이 공개한 경우):
  - 내 답, 정답, 맞음/틀림
  - 선생님이 비공개로 설정했으면 "정답은 비공개입니다" 표시

### 5-4. 재시험

- 불합격이고 선생님이 재시험을 허용했으면 **재시험 보기** 버튼이 나타납니다
- 재시험은 처음부터 다시 응시합니다
- 최대 응시 횟수가 정해져 있을 수 있습니다

### 5-5. ⭐ 시험과 클리닉의 관계

**이 부분이 중요합니다!**

```
시험에서 불합격 → 자동으로 "클리닉 대상자"가 됨
    ↓
클리닉 예약 → 보충 수업 참석 → 요건 해소
```

**클리닉 대상이 되는 경우:**
- 시험 점수가 합격점보다 낮을 때 (예: 합격점 60점인데 40점 받음)
- OMR 답안지의 마킹이 불분명할 때
- 같은 문제를 여러 번 응시해도 계속 틀릴 때
- 수업 영상을 충분히 보지 않았을 때 (90% 미만)

**클리닉 대상이 되면:**
- 홈 화면에 "클리닉 대상" 알림이 표시됩니다
- 클리닉 인증 패스에 해당 차시가 "불합격"으로 표시됩니다
- 클리닉을 예약하면 요건이 해소됩니다

**하지만 주의!**
- 예약을 취소하면 다시 클리닉 대상이 됩니다
- 예약했는데 출석하지 않으면(미출석) 다시 클리닉 대상이 됩니다

## 6. 성적 확인

**홈 → 성적** 또는 **더보기 → 성적**

### 6-1. 시험 성적
- 응시한 모든 시험의 결과가 표시됩니다
- 각 시험: 시험명, 강의명, 점수, 합격/불합격 배지
- 시험을 터치하면 문항별 상세 결과를 볼 수 있습니다

### 6-2. 과제 성적
- 아래쪽에 과제별 점수가 표시됩니다
- 합격/불합격 상태 확인

### 6-3. 정답 확인
- 선생님이 **정답 공개**로 설정한 시험만 정답을 볼 수 있습니다
- 비공개 시험은 점수만 보이고 "정답은 비공개입니다" 메시지가 표시됩니다

## 7. 과제/성적표 제출

**홈 → 제출** 또는 **더보기 → 제출**

### 7-1. 성적표 제출
1. **성적표 제출** 선택
2. 성적표 사진을 찍거나 파일을 선택합니다
3. **제출** 버튼을 누릅니다
4. 성공하면 "제출되었습니다" 메시지가 표시됩니다

### 7-2. 과제 제출
1. **과제 제출** 선택
2. 사진이나 영상 파일을 선택합니다
3. **제출** 버튼을 누릅니다

### 7-3. 내 보관함
- 제출한 모든 파일이 보관됩니다
- 폴더를 만들어 정리할 수 있습니다
- 파일 업로드, 다운로드, 삭제 가능
- 최대 파일 크기: 100MB

## 8. 클리닉 (보충 수업)

### 8-1. 클리닉이란?

클리닉은 **시험에서 불합격하거나 학습이 부족한 학생을 위한 보충 수업**입니다.

**흐름:**
```
시험 불합격 → 클리닉 대상자로 자동 지정
    ↓
클리닉 세션 확인 → 예약 → 보충 수업 참석
    ↓
클리닉 완료 → 다음 단계로 진행
```

### 8-2. 예약하기

**홈 → 클리닉** 또는 **더보기 → 클리닉**

**예약 탭:**
1. 달력에서 날짜를 선택합니다
   - 🟢 초록: 자리 있음
   - 🟡 노랑: 거의 마감
   - 🔴 빨강: 마감 (예약 불가)
2. 가능한 시간대가 아래에 표시됩니다
   - 시간, 장소, 잔여 인원 표시
   - "정원 마감"이면 선택할 수 없습니다
3. 원하는 시간대를 터치합니다
4. 메모를 입력할 수 있습니다 (선택)
   - 예: "수학 시험 보충 필요합니다"
5. **예약** 버튼을 누릅니다

**예약 후:**
- 학원 설정에 따라 즉시 확정되거나, 선생님 승인 후 확정됩니다
- 확정되면 **내 일정** 탭에서 확인할 수 있습니다

> ⚠️ **같은 세션에 중복 예약은 불가능합니다.** 이미 예약했다면 "이미 예약된 학생입니다" 메시지가 표시됩니다.

> ⚠️ **정원이 찼으면 예약할 수 없습니다.** 다른 시간대를 선택하세요.

### 8-3. 내 일정 확인

**클리닉 → 내 일정 탭**

| 상태 | 의미 | 취소 가능? |
|------|------|-----------|
| 대기 중 | 선생님 승인 전 | ✅ 취소 가능 |
| 확정됨 | 승인 완료, 참석 예정 | ❌ 선생님에게 요청 |

- 대기 중인 예약: **취소** 버튼으로 취소 가능
- 확정된 예약: 직접 취소 불가 → 선생님에게 요청

### 8-4. 클리닉 인증 패스

**더보기 → 클리닉 인증 패스**

- 클리닉 교실에 들어갈 때 이 화면을 선생님에게 보여주세요
- 화면에 표시되는 정보:
  - 내 이름, 프로필 사진
  - 실시간 시계 (매초 업데이트)
  - 차시별 합격/불합격 이력 (✅ / ❌)
  - 현재 상태: "합격" 또는 "불합격"
- 배경색이 매초 바뀌어서 **스크린샷이나 녹화로 위조할 수 없습니다**

### 8-5. 클리닉 요건 해소 규칙 (중요!)

| 상황 | 클리닉 대상 상태 |
|------|-----------------|
| 시험 불합격 | ❌ 클리닉 대상이 됨 |
| 클리닉 예약 완료 | ✅ 해소됨 (대상에서 제외) |
| 예약 취소 | ❌ 다시 클리닉 대상 |
| 당일 미출석 | ❌ 다시 클리닉 대상 |
| 정상 출석 | ✅ 해소 유지 |

## 9. 커뮤니티

**하단 탭 → 커뮤니티**

5개 탭으로 구성:

### 9-1. 공지사항
- 학원 공지를 확인합니다
- 전체공지 / 강의별 공지 / 차시별 공지로 구분
- 터치하면 전체 내용을 볼 수 있습니다

### 9-2. 게시판
- 자유 게시판 글을 읽을 수 있습니다

### 9-3. 자료실
- 학습 자료(PDF, 이미지 등)를 다운로드합니다

### 9-4. QnA (질문하기)
1. **QnA** 탭으로 이동합니다
2. **질문하기** 버튼을 누릅니다
3. 제목과 내용을 입력합니다
   - 글꼴 서식(굵게, 이탤릭 등) 사용 가능
   - 이미지 첨부 가능
4. **질문 보내기**를 누릅니다
5. "질문이 등록되었습니다" 메시지가 표시됩니다

**답변 확인:**
- 선생님이 답변하면 **알림** 탭에 알림이 옵니다
- QnA 탭에서 내 질문을 터치하면 답변을 볼 수 있습니다
- 답변이 있는 질문에는 "답변완료" 배지가 표시됩니다

### 9-5. 상담 신청
1. **상담 신청** 탭으로 이동합니다
2. 제목과 내용을 입력합니다
   - 예: "수업 시간 변경 요청", "학습 상담 요청"
3. **보내기**를 누릅니다
4. "상담 요청이 등록되었습니다" 메시지가 표시됩니다
5. 선생님 답변은 같은 곳에서 확인할 수 있습니다

## 10. 알림

**하단 탭 → 알림**

최근 7일간의 알림이 표시됩니다:
- 📋 클리닉 예약 승인됨
- 💬 QnA 질문에 답변이 달렸습니다
- 📢 새 공지사항

알림을 터치하면 해당 페이지로 바로 이동합니다.

## 11. 프로필

**더보기 → 프로필**

### 11-1. 내 정보 수정
- 오른쪽 위 **편집** 버튼을 누릅니다
- 수정 가능: 이름, 전화번호, 성별, 학교, 학년, 반, 주소
- **저장** 버튼으로 저장합니다

### 11-2. 프로필 사진 변경
- 프로필 사진 영역을 터치합니다
- 사진을 선택합니다 (최대 5MB)

### 11-3. 아이디 변경
- **아이디 변경하기**를 터치합니다
- 새 아이디를 입력합니다
- 이미 사용 중인 아이디는 사용할 수 없습니다

### 11-4. 비밀번호 변경
- **비밀번호 변경하기**를 터치합니다
- 현재 비밀번호와 새 비밀번호를 입력합니다
- **변경** 버튼을 누릅니다

## 12. 보관함 (내 인벤토리)

**더보기 → 내 인벤토리**

- 제출한 파일과 선생님이 공유한 자료가 보관됩니다
- **새 폴더**: 폴더를 만들어 정리
- **업로드**: 파일 추가 (최대 100MB)
- 파일을 터치하면 다운로드
- 삭제: 파일 선택 후 삭제 (확인 팝업)

## 13. 출결 현황

**더보기 → 출결 현황**

- 출석 정보는 각 수업의 차시 상세에서 확인할 수 있습니다
- **수업 목록 보기**를 터치하면 일정 페이지로 이동합니다

## 14. 설정 / 로그아웃

**더보기 → 로그아웃**: 로그아웃 후 로그인 화면으로 돌아갑니다

## 15. 자주 묻는 질문

### 비밀번호를 잊었어요
로그인 화면 → **비밀번호 찾기** → 전화번호 입력 → 카카오톡으로 인증번호 → 임시 비밀번호 발급

### 시험 답을 실수로 제출했어요
- 선생님이 재시험을 허용했으면 **재시험 보기** 버튼이 나타납니다
- 허용되지 않았으면 선생님에게 직접 문의하세요

### 시험에서 떨어졌는데 클리닉이 뭔가요?
- 클리닉은 **보충 학습 시간**입니다
- 시험 불합격 시 자동으로 클리닉 대상이 됩니다
- 클리닉 탭에서 예약하고 참석하면 요건이 해소됩니다
- 예약 후 취소하거나 미출석하면 다시 대상이 됩니다

### 클리닉 예약을 취소하고 싶어요
- 아직 승인 전(대기 중)이면 **내 일정**에서 취소할 수 있습니다
- 이미 승인된 예약은 선생님에게 취소를 요청하세요

### 영상이 안 재생돼요
- 인터넷 연결을 확인하세요
- 앱을 새로고침(위에서 아래로 당기기)해 보세요
- 계속 안 되면 선생님에게 문의하세요

### 화면이 이상하게 보여요
- "업데이트가 있습니다" 메시지가 나오면 **새로고침** 버튼을 누르세요
- 그래도 이상하면 브라우저 캐시를 삭제하고 다시 접속하세요
