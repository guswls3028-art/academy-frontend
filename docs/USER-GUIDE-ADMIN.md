# 선생님/관리자 사용 가이드

## 시작하기

### 로그인
1. 학원 전용 주소(예: tchul.com)에 접속합니다
2. **로그인** 버튼을 누릅니다
3. 아이디와 비밀번호를 입력 후 **로그인**을 누릅니다
4. 대시보드가 나타나면 시작할 수 있습니다

### 화면 구성
- **왼쪽 사이드바**: 모든 메뉴가 있습니다
- **상단 헤더**: 알림, 근무 시간, 프로필
- **가운데**: 선택한 메뉴의 내용이 표시됩니다

---

## 대시보드

로그인하면 가장 먼저 보이는 화면입니다.

**한눈에 볼 수 있는 정보:**
- 미답변 질문 수
- 대기 중인 학생 제출물
- 진행 중인 시험
- 알림톡 잔액

각 항목을 클릭하면 해당 페이지로 바로 이동합니다.

---

## 학생 관리

**사이드바 → 학생**

### 학생 목록 보기
- 모든 학생이 표 형태로 표시됩니다
- **검색창**에 이름, 전화번호, 학교명을 입력하면 바로 찾을 수 있습니다
- 학생 이름을 클릭하면 상세 정보가 오른쪽에 나타납니다

### 학생 등록
1. 오른쪽 상단 **학생 등록** 버튼을 누릅니다
2. **1명만 등록** 또는 **엑셀 업로드** 중 선택합니다

**1명 등록 시:**
- 이름, 초기 비밀번호, 학부모 전화번호를 입력합니다
- **가입 안내 알림톡 발송** 토글을 켜면 학생과 학부모에게 로그인 정보가 카카오톡으로 전송됩니다
- **등록**을 누르면 완료됩니다

**엑셀 업로드 시:**
- **엑셀 양식 다운로드**를 눌러 양식을 받습니다
- 필수: 이름, 학부모전화번호 (나머지는 선택)
- 작성한 파일을 업로드하고 초기 비밀번호를 입력합니다
- **등록**을 누르면 백그라운드에서 처리됩니다

### 학생 관리 기능
학생을 선택(체크)한 후 상단 버튼으로:
- **메시지 발송**: 선택한 학생/학부모에게 알림톡 발송
- **엑셀 다운로드**: 학생 정보 내보내기
- **태그 추가**: 학생 분류용 태그 부여
- **삭제**: 30일간 보관 후 자동 삭제 (복원 가능)

## 강의 관리

**사이드바 → 강의**

### 강의 만들기
1. **+ 강의 추가** 버튼을 누릅니다
2. 강의명, 과목, 담당 강사를 입력합니다
3. 운영 기간과 수업 시간을 설정합니다
4. 색상을 선택합니다 (시간표에서 구분하기 위함)
5. **생성**을 누릅니다

### 수강생 등록
1. 강의 카드를 클릭합니다
2. **수강생** 탭으로 이동합니다
3. **학생 추가** 버튼을 누릅니다
4. 기존 학생 목록에서 선택하거나, 엑셀로 일괄 등록합니다

### 차시(수업) 추가
1. 강의 카드를 클릭합니다
2. **차시** 영역에서 **+** 버튼을 누릅니다
3. 날짜와 시간이 자동으로 채워집니다
4. **보강** 차시도 별도로 추가할 수 있습니다

### 강의 종료/삭제
- 강의 카드의 **⚙️(설정)** 버튼을 누릅니다
- **강의 종료**: "지난강의" 탭으로 이동됩니다 (언제든 복원 가능)
- **강의 삭제**: 모든 데이터가 영구 삭제됩니다 (확인 팝업 표시)

---

## 출결 관리

**강의 → 차시 선택 → 출결 탭**

### 출결 입력
- 학생 이름 옆 상태 배지를 클릭합니다
- 상태를 선택합니다:
  - **현장**: 직접 출석
  - **영상**: 온라인 수업 참여
  - **보강**: 보강 수업 출석
  - **지각**: 지각 출석
  - **결석**: 미출석
  - **조퇴**: 조기 퇴실

### 일괄 출석 처리
- **전체 출석** 버튼을 누르면 모든 학생이 "현장"으로 설정됩니다

---

## 시험 관리

**사이드바 → 시험** 또는 **차시 → 시험 탭**

### 시험 만들기
1. 차시를 선택한 후 **시험 탭**으로 이동합니다
2. **시험 추가** 버튼을 누릅니다
3. 시험명, 총점, 합격점을 입력합니다
4. 문항을 추가합니다 (객관식/주관식)
5. 정답과 배점을 설정합니다

### 시험 종료
- 시험 상단의 **종료** 버튼을 누릅니다
- 종료 후에는 학생이 더 이상 응시할 수 없습니다
- 확인 팝업이 표시됩니다

### 정답 공개 설정
- 시험 설정에서 **정답 공개** 여부를 선택합니다
- 비공개로 설정하면 학생은 점수만 확인할 수 있고 정답은 보이지 않습니다

---

## 성적 입력

**차시 → 성적 탭**

### 성적 입력 방법
1. 차시를 선택한 후 **성적** 탭으로 이동합니다
2. **편집** 버튼을 누릅니다
3. 각 학생의 점수를 직접 입력합니다
   - Tab/화살표 키로 빠르게 이동할 수 있습니다
   - 자동 저장됩니다 (중간에 나가도 복원 가능)
4. **저장** 버튼을 누릅니다

### 성적 내보내기
- **엑셀 다운로드** 버튼으로 성적표를 내보낼 수 있습니다
- **인쇄** 버튼으로 성적표를 인쇄할 수 있습니다

---

## 영상 관리

**사이드바 → 영상**

### 영상 업로드
1. 왼쪽 폴더 트리에서 강의/차시를 선택합니다
2. **영상 업로드** 버튼을 누릅니다
3. 파일을 선택합니다 (MP4, MOV 등)
4. 업로드가 시작되면 진행률이 표시됩니다
5. 업로드 완료 후 자동으로 인코딩이 시작됩니다
6. 인코딩 완료 후 학생이 시청할 수 있습니다

### 영상 상태
- **처리 중**: 인코딩 진행 중 (잠시 기다려 주세요)
- **완료**: 학생이 시청 가능
- **실패**: 오류 발생 (재시도 버튼으로 다시 시도)

### 영상 상세
- 영상 카드를 클릭하면 상세 정보가 표시됩니다
- 조회수, 좋아요, 댓글을 확인할 수 있습니다
- 학생별 시청 현황을 확인할 수 있습니다
- 삭제 버튼으로 영상을 삭제할 수 있습니다

---

## 커뮤니티

**사이드바 → 커뮤니티**

### 공지사항 작성
1. **공지사항** 탭으로 이동합니다
2. **+ 추가** 버튼을 누릅니다
3. 제목과 내용을 입력합니다
4. **등록**을 누르면 학생들에게 즉시 공개됩니다

### QnA 답변
1. **QnA** 탭으로 이동합니다
2. 왼쪽 목록에서 학생 질문을 선택합니다
3. 아래쪽 답변 입력란에 답변을 작성합니다
4. **답변 등록**을 누릅니다 (또는 Ctrl+Enter)
5. 학생에게 알림이 전송됩니다

### 상담 처리
1. **상담** 탭으로 이동합니다
2. 학생 상담 요청을 선택합니다
3. 답변을 작성하고 등록합니다

### 자료실
- **자료실** 탭에서 학습 자료(PDF, 이미지 등)를 업로드합니다
- 학생들이 자료실에서 다운로드할 수 있습니다

---

## 클리닉 (보충 학습)

**사이드바 → 클리닉**

### 클리닉 세션 만들기
1. **일정 관리** 탭에서 캘린더의 날짜를 선택합니다
2. 시간, 장소, 정원, 제목을 입력합니다
3. **클리닉 만들기**를 누릅니다
4. 대상 학년이나 강의를 지정할 수 있습니다

### 학생 배정
- **예약** 탭에서 학생을 수동으로 세션에 배정할 수 있습니다
- 학생이 직접 예약한 경우 목록에 자동으로 나타납니다
- 자동 승인이 꺼져 있으면 **승인/거절** 처리가 필요합니다

### 출석 확인
1. **클리닉 홈** 또는 **클리닉 진행** 탭에서 오늘 세션을 확인합니다
2. 학생 이름 옆 출석 버튼을 눌러 상태를 변경합니다
   - 출석 / 미출석

### 클리닉 패스카드
- **설정** 탭에서 패스카드 배경색을 변경할 수 있습니다
- 학생이 앱에서 패스카드를 보여주면 색상으로 본인 확인이 가능합니다
- 매일 색상을 자동 변경하는 옵션도 있습니다

---

## 메시지 관리

**사이드바 → 메시지**

### 학생/학부모에게 메시지 보내기
1. **학생 목록**에서 대상 학생을 선택합니다
2. **메시지 발송** 버튼을 누릅니다
3. 발송 방법을 선택합니다: 알림톡(카카오) / 문자(SMS)
4. 템플릿을 선택하거나 직접 작성합니다
5. **발송**을 누릅니다

### 자동 발송 설정
**메시지 → 자동발송** 탭에서 아래 상황에 자동 메시지를 설정할 수 있습니다:
- 학생 가입 완료 시
- 수업 시작 전 알림
- 시험 일정 안내
- 성적 공개 시
- 결석 발생 시
- 클리닉 예약 확인

### 메시지 발송 기록
- **로그** 탭에서 모든 발송 이력을 확인할 수 있습니다
- 성공/실패 상태, 수신자, 발송 시간이 기록됩니다

---

## 직원 관리

**사이드바 → 직원관리**

### 직원 추가
1. **+ 직원 추가** 버튼을 누릅니다
2. 이름, 전화번호, 역할(강사/조교)을 입력합니다
3. 시급 태그를 지정합니다

### 근무 시간 기록
- 상단 헤더의 **출근/퇴근** 버튼으로 근무 시간을 기록합니다
- **휴식** 버튼으로 휴식 시간을 분리할 수 있습니다
- 실시간 경과 시간이 표시됩니다

### 급여 확인
- **보고** 탭에서 월별 급여 내역을 확인합니다
- 직원별 근무 시간, 시급, 총 급여가 계산됩니다

---

## 저장소

**사이드바 → 저장소**

- 학원 자료를 폴더별로 정리할 수 있습니다
- 파일을 드래그&드롭으로 업로드합니다
- 학생 인벤토리도 이곳에서 확인할 수 있습니다

---

## 도구

**사이드바 → 도구**

### PPT 생성기
1. 이미지 또는 PDF 파일을 업로드합니다
2. 비율(16:9 또는 4:3), 배경색, 밝기를 설정합니다
3. **생성** 버튼을 누릅니다
4. 생성이 완료되면 자동으로 PPT 파일이 다운로드됩니다

---

## 설정

**사이드바 → 설정**

### 내 정보
- 이름, 전화번호 변경
- 비밀번호 변경
- 로그아웃

### 학원 정보 (대표만)
- 학원명, 연락처, 주소 수정
- 카카오톡 미리보기(OG) 설정
- 사업자 정보 입력

### 테마
- 화면 색상 테마를 변경할 수 있습니다
- 12가지 프리셋 중 선택

### 결제/구독
- 현재 구독 상태, 요금제, 남은 기간 확인
- 갱신 문의

## 자주 하는 작업 순서

### 새 학기 준비
1. 학생 등록 (엑셀 업로드)
2. 강의 생성
3. 수강생 등록
4. 차시 추가
5. 학습 자료 업로드

### 매 수업마다
1. 출결 입력
2. 성적 입력 (시험/과제가 있는 경우)
3. QnA 답변 확인

### 시험 운영
1. 시험 생성 + 문항 입력
2. 학생에게 공지
3. 응시 현황 확인
4. 채점 (객관식 자동, 주관식 수동)
5. 성적 공개

### 클리닉 운영
1. 세션 생성 (날짜/시간/장소)
2. 대상 학생 배정 또는 학생 자율 예약
3. 당일 출석 확인
4. 보충 학습 진행
