# 선생님/관리자 사용 가이드

---

## 1. 시작하기

### 1-1. 로그인
1. 학원 전용 주소(예: tchul.com)에 접속합니다
2. **로그인** 버튼을 누릅니다
3. 아이디와 비밀번호를 입력합니다
4. **로그인**을 누르면 대시보드가 나타납니다

> 💡 비밀번호를 잊었다면 **비밀번호 찾기**를 누르세요. 등록된 전화번호로 임시 비밀번호가 카카오톡으로 전송됩니다.

### 1-2. 화면 구성
- **왼쪽 사이드바**: 모든 메뉴 (접기/펼치기 가능)
- **상단 헤더**: 근무 시간 타이머, 알림 벨, 프로필
- **가운데**: 선택한 메뉴의 내용

### 1-3. 대시보드
로그인 후 첫 화면입니다.

| 항목 | 설명 |
|------|------|
| 미답변 질문 | 학생이 보낸 QnA 중 아직 답변하지 않은 수 |
| 대기 제출물 | 학생이 제출한 성적표/과제 중 처리 대기 건수 |
| 활성 시험 | 현재 진행 중인 시험 수 |
| 알림톡 잔액 | 카카오 알림톡/문자 발송 가능 잔여 건수 |

각 숫자를 클릭하면 해당 페이지로 바로 이동합니다.

## 2. 학생 관리

### 2-1. 학생 등록

**1명씩 등록하기:**
1. 사이드바 **학생** → 오른쪽 상단 **학생 등록**
2. **1명만 등록** 선택
3. 다음 정보를 입력합니다:

| 항목 | 필수 | 설명 |
|------|------|------|
| 이름 | ✅ | 학생 이름 |
| 초기 비밀번호 | ✅ | 학생이 첫 로그인에 사용할 비밀번호 |
| 학부모 전화번호 | ✅ | 010으로 시작하는 11자리. 학부모 아이디 겸 연락처 |
| 학생 전화번호 | 선택 | 입력하면 학생 아이디로 사용. 비워두면 자동 부여 |
| 성별 | 선택 | 남/여 |
| 학교/학년/반 | 선택 | XX고, XX중 입력 시 학교유형 자동 판별 |

4. **가입 안내 알림톡 발송** 토글:
   - **켜면**: 학생 전화번호와 학부모 전화번호로 카카오 알림톡이 발송됩니다
   - 알림톡 내용: 아이디, 비밀번호, 접속 링크
   - 학부모에게는 학부모 아이디(=학부모 전화번호), 학부모 비밀번호(0000), 학생 아이디/비밀번호가 함께 전송됩니다
5. **등록** 버튼을 누르면 완료

> 💡 학부모 아이디는 **학부모 전화번호** 그 자체이며, 초기 비밀번호는 **0000**입니다.

**엑셀로 일괄 등록하기:**
1. **엑셀 업로드** 선택
2. **엑셀 양식 다운로드**를 눌러 양식 파일을 받습니다
3. 양식에 학생 정보를 작성합니다 (필수: 이름, 학부모전화번호)
4. 작성한 파일을 업로드합니다
5. 초기 비밀번호를 입력합니다 (모든 학생에 동일 적용)
6. **등록**을 누르면 백그라운드에서 처리됩니다
7. 오른쪽 아래 진행 상태 바에서 처리 현황을 확인할 수 있습니다

### 2-2. 학생 목록

- **검색**: 이름, 전화번호, 학교명으로 검색
- **정렬**: 이름, 등록일, 학교 등 기준으로 정렬
- **컬럼 설정**: 표에 보이는 열을 원하는 대로 선택
- **학생 클릭** → 오른쪽에 상세 패널이 열립니다:
  - 기본 정보, 수강 강의, 태그, 메모
  - 수정, 삭제, 메시지 발송, 비밀번호 초기화 가능

### 2-3. 학생 대량 작업

학생을 체크박스로 선택한 후 상단 도구 모음:

| 버튼 | 동작 |
|------|------|
| 메시지 발송 | 선택한 학생/학부모에게 카카오 알림톡 또는 문자 발송 |
| 엑셀 다운로드 | 선택한 학생 정보를 엑셀 파일로 내보내기 |
| 태그 추가 | 분류용 태그(색상 라벨) 일괄 부여 |
| 삭제 | 소프트 삭제 → 30일간 "삭제됨" 탭에 보관 후 자동 삭제 |

### 2-4. 가입 신청 관리

학생이 직접 회원가입한 경우:
1. 사이드바 **학생** → **가입 신청** 탭
2. 대기 중인 신청 목록이 표시됩니다
3. **승인**: 학생 계정이 생성되고 알림톡이 자동 발송됩니다
4. **거절**: 신청이 거절됩니다

> 💡 학원 설정에서 **자동 승인**을 켜면 학생 가입 신청 시 바로 승인됩니다.

## 3. 강의 관리

### 3-1. 강의 생성
1. 사이드바 **강의** → **+ 강의 추가**
2. 입력 항목:

| 항목 | 필수 | 설명 |
|------|------|------|
| 강의명 | ✅ | 예: "수학 기초반", "영어 심화반" |
| 과목 | ✅ | 예: 수학, 영어, 과학 |
| 담당 강사 | 선택 | 직원 목록에서 선택 |
| 운영 기간 | 선택 | 시작일~종료일 |
| 수업 시간 | 선택 | 예: "화,목 14:00~16:00" |
| 색상 | 선택 | 시간표/카드에서 강의 구분용 |

3. **생성** 버튼을 누르면 완료

### 3-2. 수강생 관리

강의 카드를 클릭하면 해당 강의의 상세 페이지가 열립니다.

**수강생 추가:**
1. **수강생** 탭으로 이동
2. **학생 추가** 버튼 클릭
3. 세 가지 방법:
   - **기존 학생 선택**: 등록된 학생 목록에서 검색하여 추가
   - **신규 학생 등록**: 학생 등록 모달에서 바로 등록 후 추가
   - **엑셀 업로드**: 수강생 명단 엑셀로 일괄 등록

**수강생 목록에서 볼 수 있는 정보:**
- 학생 이름, 전화번호
- 차시별 출결 현황 (매트릭스 형태)
- 메시지 발송, 엑셀 다운로드 가능

### 3-3. 차시(수업) 관리

**차시 추가:**
1. 차시 바에서 **+** 버튼 클릭
2. 자동으로 다음 회차 번호, 날짜(+7일), 시간(강의 시간)이 채워집니다
3. **보강 차시**: 별도로 추가 가능 (날짜/시간 직접 입력)

**차시를 선택하면 5개 탭이 나타납니다:**

| 탭 | 설명 |
|------|------|
| **출결** | 학생별 출석/지각/결석 등 상태 관리 |
| **성적** | 시험/과제 점수 입력 및 조회 |
| **시험** | 시험 생성, 문항 관리, 학생 응시 현황 |
| **과제** | 과제 생성, 제출 현황, 채점 |
| **영상** | 수업 영상 업로드 및 관리 |

### 3-4. 강의 종료/복원/삭제

강의 카드의 **⚙️(설정)** 클릭:
- **강의 종료**: "지난강의" 탭으로 이동. 데이터 보존. 언제든 복원 가능
  - 확인 팝업: "강의를 종료하시겠습니까? 종료된 강의는 '지난강의' 탭에서 다시 복원할 수 있습니다."
- **강의 복원**: 지난강의 탭에서 다시 운영 중으로 돌림
- **강의 삭제**: 모든 데이터(차시, 수강생, 출결, 성적) 영구 삭제
  - 확인 팝업: "모든 관련 정보가 제거됩니다"

## 4. 출결 관리

### 4-1. 출결 입력

**강의 → 차시 선택 → 출결 탭**

| 출결 상태 | 의미 | 학생앱 표시 |
|----------|------|-----------|
| 현장 | 직접 출석 | 현 |
| 영상 | 온라인 수업 참여 | 영 |
| 보강 | 보강 수업 출석 | 보 |
| 지각 | 지각 출석 | 지 |
| 조퇴 | 조기 퇴실 | 조 |
| 결석 | 미출석 | 결 |
| 출튀 | 출석 후 무단 이탈 | 출 |
| 자료 | 자료만 제공 | 자 |
| 부재 | 장기 부재 | 부 |
| 퇴원 | 퇴원 처리 | 퇴 |

**조작 방법:**
- 학생 이름 옆 상태 배지를 클릭 → 팝오버에서 상태 선택
- **전체 출석** 버튼: 모든 학생을 "현장"으로 일괄 변경
- 메모: 각 학생별 출결 메모 입력 가능

### 4-2. 출결 매트릭스

**강의 상세 → 수강생 탭**에서 전체 차시의 출결 현황을 한눈에 볼 수 있습니다.
- 행: 학생
- 열: 차시(1차시, 2차시, ...)
- 셀: 출결 상태 약자(현/영/결/지...)

## 5. 시험 관리

### 5-1. 시험 생성

**차시 → 시험 탭 → 시험 추가**

| 항목 | 설명 |
|------|------|
| 시험명 | 예: "1차 중간고사", "단원평가 3장" |
| 총점 | 시험 만점 (예: 100) |
| 합격점 | 이 점수 이상이면 합격 (예: 60) |
| 재시험 허용 | 켜면 학생이 불합격 시 재응시 가능 |
| 최대 응시 횟수 | 재시험 포함 총 응시 가능 횟수 |
| 정답 공개 | 학생에게 정답을 보여줄지 여부 |

**문항 추가:**
- 객관식: 5지선다 (정답 번호 지정)
- 주관식: 단답형 (정답 텍스트)
- 서술형: 교사가 수동 채점

**정답지 등록:**
- 정답 번호를 한 번에 입력할 수 있습니다 (클릭 한 번으로 붙여넣기)
- 배점도 문항별로 설정 가능

### 5-2. 시험 운영

시험 상태:
| 상태 | 의미 |
|------|------|
| 초안 (DRAFT) | 작성 중, 학생에게 미공개 |
| 진행 중 (OPEN) | 학생이 응시 가능 |
| 종료 (CLOSED) | 응시 마감, 더 이상 제출 불가 |

**시험 종료:**
- 시험 상단 **종료** 버튼 클릭
- 확인 팝업: "시험을 종료하시겠습니까? 종료 이후엔 답안 제출이 불가합니다."
- 종료 후에는 학생이 더 이상 응시할 수 없습니다

### 5-3. 채점

- **객관식**: 정답지 기준으로 자동 채점
- **주관식**: 교사가 직접 점수 입력
- 채점 결과: 합격/불합격 자동 판정 (합격점 기준)

## 6. 성적 관리

### 6-1. 성적 입력 (엑셀식)

**차시 → 성적 탭**

1. **편집** 모드 진입
2. 표 형태로 학생별 시험/과제 점수를 직접 입력
   - Tab, 화살표 키로 셀 간 이동
   - 입력 즉시 자동 임시 저장 (중간에 나가도 복원 가능)
3. **저장** 버튼으로 확정

**추가 기능:**
- 엑셀 다운로드: 성적표를 파일로 내보내기
- 인쇄: 성적표 인쇄 (클리닉 현황 포함)
- 학생 이름 클릭 → 오른쪽 드로어에서 개별 성적 상세, 시험 이력 확인

### 6-2. ⭐ 성적과 클리닉의 연결 (핵심!)

**이것이 시스템의 핵심 흐름입니다:**

```
시험 응시/채점 → 성적 계산 → 불합격 판정 → 클리닉 대상자 자동 지정
                                              ↓
                              교사 확인 → 클리닉 세션 배정 → 학생 예약
                                              ↓
                              학생 출석 → 클리닉 요건 해소 → 다음 차시 진행
```

**자동 클리닉 대상자 지정 조건:**

성적이 저장되면 시스템이 자동으로 다음을 확인합니다:

| 조건 | 설명 | 예시 |
|------|------|------|
| **시험 불합격** | 시험 점수 < 합격점 | 40점 / 합격점 60점 → 클리닉 대상 |
| **OMR 신뢰도 낮음** | OMR 인식 시 2개 이상 문항이 확신도 부족 | 마킹이 불분명한 답안 |
| **반복 오답** | 같은 문항을 2회 이상 응시에서 연속 틀림 | 재시험에서도 같은 문제를 틀림 |
| **차시 미이수** | 영상 시청 부족(<90%), 시험 불합격, 과제 미제출 | 종합적으로 차시 완료 조건 불충족 |

**이 중 하나라도 해당하면 학생은 자동으로 "클리닉 대상자"가 됩니다.**

## 7. 클리닉 관리 (보충 학습)

### 7-1. 전체 흐름

```
① 시험/과제 채점 → 불합격 학생 자동 감지
     ↓
② 교사: 클리닉 → 예약 탭에서 "미예약 대상자" 목록 확인
     ↓
③ 교사: 클리닉 세션 생성 (날짜, 시간, 장소, 정원)
     ↓
④ 학생 배정: 교사가 수동 배정 또는 학생이 직접 예약
     ↓
⑤ 클리닉 당일: 교사가 출석 확인
     ↓
⑥ 클리닉 요건 해소: 학생이 예약하는 순간 자동으로 해소됨
     ↓
⑦ 취소/미출석 시: 클리닉 요건이 다시 살아남 → 재예약 필요
```

### 7-2. 클리닉 대상자 확인

**클리닉 → 예약 탭**

- **클리닉 대상자** 테이블이 표시됩니다
- 각 대상자 정보:

| 항목 | 의미 |
|------|------|
| 학생 이름 | 클리닉이 필요한 학생 |
| 차시명 | 어떤 수업에서 미달됐는지 |
| 사유 | "점수 미달" 또는 "OMR 신뢰도" |
| 시험 점수 | 실제 받은 점수 |
| 합격 기준점 | 넘어야 하는 점수 |
| 예약 상태 | 미예약 / 예약됨 |

- **미예약 학생**을 선택한 후 클리닉 세션에 배정할 수 있습니다

### 7-3. 클리닉 세션 생성

**클리닉 → 일정 관리 탭**

1. 캘린더에서 날짜를 선택합니다
2. 입력 항목:

| 항목 | 필수 | 설명 |
|------|------|------|
| 제목 | 선택 | 예: "수학 보충", "영어 재시험" |
| 시간 | ✅ | 시작 시간 (예: 14:00) |
| 시간(분) | ✅ | 소요 시간 (기본 60분) |
| 장소 | ✅ | 예: "보충학습실 A", "3층 교실" |
| 정원 | ✅ | 최대 참가 인원 |
| 대상 학년 | 선택 | 전체/1학년/2학년/3학년 |
| 대상 학교유형 | 선택 | 전체/고등/중등 |
| 대상 강의 | 선택 | 특정 강의 수강생만 |

3. **클리닉 만들기** 버튼 클릭
4. 성공 메시지 확인

> 💡 **일괄 생성**: 여러 날짜를 선택하면 같은 설정으로 한 번에 여러 세션을 만들 수 있습니다 (최대 20일).

### 7-4. 학생 배정 방식

**방법 1: 교사가 수동 배정**
- 클리닉 대상자 목록에서 학생을 선택
- 세션에 배정하면 바로 **"예약됨(booked)"** 상태가 됩니다
- 학생 앱의 "내 일정"에 즉시 반영됩니다

**방법 2: 학생이 직접 예약**
- 학생이 앱에서 클리닉 → 예약 탭에서 세션을 선택하고 예약합니다
- **자동 승인 설정**에 따라:
  - 자동 승인 **켜짐**: 예약 즉시 확정 (booked)
  - 자동 승인 **꺼짐**: "대기(pending)" 상태 → 교사가 승인/거절 필요

### 7-5. 클리닉 요건 해소 규칙

| 상황 | 클리닉 요건 상태 |
|------|----------------|
| 학생이 예약함 (booked) | ✅ **해소됨** — 클리닉 대상에서 제외 |
| 교사가 수동 배정 | ✅ **해소됨** |
| 학생이 예약 취소 | ❌ **다시 미해소** — 클리닉 대상으로 복귀 |
| 교사가 "미출석" 처리 | ❌ **다시 미해소** — 클리닉 대상으로 복귀 |
| 학생이 출석함 (attended) | ✅ 해소 유지 |

> ⚠️ **핵심**: 예약만 해도 요건은 해소됩니다. 하지만 취소하거나 미출석이면 다시 대상자가 됩니다.

### 7-6. 클리닉 홈 (당일 운영)

**클리닉 → 홈**

- 오늘 예정된 클리닉 세션이 타임라인으로 표시됩니다
- 각 세션에서:
  - 예약 학생 목록 확인
  - **출석/미출석** 상태 변경 (버튼 클릭)
  - 메모 작성
- **대기 승인**: 학생이 신청한 예약 중 승인 대기 건이 있으면 상단에 표시됩니다

### 7-7. 클리닉 패스카드

**클리닉 → 설정(패스카드)**

- 패스카드 배경색 3가지를 설정할 수 있습니다
- **매일 자동 변경** 옵션: 켜면 매일 색상이 자동으로 바뀝니다
- 학생은 앱에서 패스카드 화면을 보여줍니다
- 패스카드에는:
  - 학생 이름, 프로필 사진
  - 실시간 시계 (매초 업데이트)
  - 차시별 합격/불합격 이력
  - 배경색 (매초 미세 변화 → 스크린샷/녹화 방지)

### 7-8. 클리닉 운영 콘솔

**클리닉 → 클리닉 진행**

- 왼쪽: 캘린더 + 세션 목록
- 오른쪽: 선택한 세션의 참가자 상세
  - 참가자별 출석 상태 토글
  - 미통과 항목(어떤 시험에서 떨어졌는지) 표시
  - 학생 이름 클릭 → 학생 상세 오버레이

## 8. 영상 관리

### 8-1. 영상 업로드

**사이드바 → 영상**

1. 왼쪽 트리에서 강의 → 차시를 선택
2. **영상 업로드** 클릭
3. 파일 선택 (MP4, MOV 등 지원)
4. 업로드 진행률이 표시됩니다
5. 업로드 완료 후 자동으로 인코딩이 시작됩니다

### 8-2. 영상 처리 상태

| 상태 | 의미 | 학생 시청 |
|------|------|----------|
| 대기 중 (PENDING) | 업로드 직후 | ❌ |
| 처리 중 (PROCESSING) | 인코딩 진행 중 | ❌ |
| 완료 (READY) | 인코딩 완료 | ✅ 시청 가능 |
| 실패 (FAILED) | 인코딩 오류 | ❌ 재시도 필요 |

- 실패 시 **재시도** 버튼으로 다시 시도할 수 있습니다
- 30분 이하 영상: 빠른 처리 (보통 수 분)
- 30분 초과 영상: 별도 처리 (최대 수 시간)

### 8-3. 영상 상세

영상 카드를 클릭하면:
- 조회수, 좋아요 수, 댓글 수
- 학생별 시청 현황 (누가 얼마나 봤는지)
- 영상 공개/비공개 설정
- 시청 제한: 자유 시청 / 1회 제한 / 차단
- 워터마크 표시 여부
- 삭제 (확인 팝업 → 6개월 보관 후 영구 삭제)

## 9. 커뮤니티

### 9-1. 공지사항

**커뮤니티 → 공지사항**

**공지 작성:**
1. **+ 추가** 버튼 클릭
2. 제목 입력
3. 내용 작성 (리치 텍스트 에디터: 굵게, 이탤릭, 링크, 이미지 삽입 가능)
4. **등록** 클릭

- 공지 범위: 전체 공지 / 특정 강의 공지 / 특정 차시 공지
- 학생 앱의 공지사항 탭에 즉시 반영됩니다

### 9-2. QnA 답변

**커뮤니티 → QnA**

1. 왼쪽에 학생 질문 목록이 표시됩니다
   - 미답변 질문에 빨간 표시
   - 삭제된 학생의 질문은 "삭제된 학생" 표시
2. 질문을 클릭하면 오른쪽에 대화 스레드가 열립니다
3. 아래쪽 입력란에 답변을 작성합니다
4. **답변 등록** 클릭 (또는 Ctrl+Enter)
5. 자동발송 설정이 켜져 있으면 학생에게 알림톡이 발송됩니다

### 9-3. 상담

**커뮤니티 → 상담**

- QnA와 같은 형태이지만 상담 요청 전용입니다
- 학생/학부모의 수업 시간 변경, 상담 요청 등을 처리합니다

### 9-4. 게시판 / 자료실

- **게시판**: 자유 게시판 관리 (글 작성, 수정, 삭제, 댓글)
- **자료실**: 학습 자료 업로드 (PDF, 이미지, 문서). 학생이 다운로드 가능

## 10. 메시지 관리

### 10-1. 직접 발송

학생 목록/수강생 목록에서 학생을 선택 → **메시지 발송**

| 발송 방식 | 설명 |
|----------|------|
| 알림톡 | 카카오톡 알림톡 (사전 승인된 템플릿만 가능) |
| 문자 (SMS) | 일반 문자 메시지 |
| 둘 다 | 알림톡 우선, 실패 시 문자로 대체 |

### 10-2. 템플릿 관리

**메시지 → 템플릿**

- 자주 사용하는 메시지를 템플릿으로 저장합니다
- 변수 사용 가능: `#{학생이름}`, `#{강의명}`, `#{시험명}` 등
- 카카오 알림톡은 검수 신청 후 승인되어야 사용 가능합니다

### 10-3. 자동 발송

**메시지 → 자동발송**

이벤트 발생 시 자동으로 메시지를 보낼 수 있습니다:

| 이벤트 | 수신 대상 | 예시 |
|--------|----------|------|
| 가입 안내 | 학생 + 학부모 | 아이디/비밀번호 안내 |
| 수업 시작 전 | 학생 + 학부모 | "오늘 수업이 시작됩니다" |
| 입실 완료 | 학부모 | "학생이 입실했습니다" |
| 결석 발생 | 학부모 | "결석이 발생했습니다" |
| 시험 예정 | 학생 | "시험이 예정되어 있습니다" |
| 성적 공개 | 학생 + 학부모 | "성적이 공개되었습니다" |
| 과제 등록 | 학생 | "새 과제가 등록되었습니다" |
| 클리닉 예약 | 학생 | "예약이 완료되었습니다" |
| 퇴원 처리 | 학생 + 학부모 | "퇴원 처리가 완료되었습니다" |

각 이벤트별로 **켜기/끄기**, 사용할 **템플릿**, **발송 시점**(즉시 / N분 전)을 설정할 수 있습니다.

### 10-4. 발송 기록

**메시지 → 로그**

- 모든 발송 이력이 기록됩니다
- 성공/실패 상태, 수신자, 발송 시간 확인
- 실패한 메시지의 원인 확인 가능

## 11. 직원 관리

### 11-1. 직원 추가

**직원관리 → + 직원 추가**

- 이름, 전화번호, 역할(강사/조교) 입력
- 프로필 사진 등록 가능
- 시급 태그(근무 유형) 지정: 예) "주말 수학", "방과후 영어"

### 11-2. 근무 시간

**상단 헤더의 시계 영역:**
- **출근**: 근무 시작
- **휴식**: 휴식 시간 시작/종료 (근무 시간에서 제외)
- **퇴근**: 근무 종료
- 실시간 경과 시간이 표시됩니다
- 현재 근무 중인 직원 목록도 확인 가능

### 11-3. 급여

**직원관리 → 보고**

- 월별 근무 시간 합계
- 시급 × 근무 시간 = 급여 자동 계산
- 급여 스냅샷 저장/내보내기

## 15. 전체 업무 흐름 요약

### 새 학기 세팅
```
학생 등록(엑셀) → 알림톡 발송 → 강의 생성 → 수강생 배정 → 차시 추가 → 영상/자료 업로드
```

### 매 수업
```
출결 입력 → 시험/과제 채점 → 성적 저장
                ↓
     불합격 학생 자동 감지 (클리닉 대상)
                ↓
     클리닉 세션 생성 → 학생 배정/예약 → 당일 출석 확인
```

### 시험 운영
```
시험 생성 → 문항 입력 → 정답지 설정 → 학생 공지(알림톡)
    ↓
학생 응시 → 자동 채점 (객관식) + 수동 채점 (주관식)
    ↓
합격/불합격 판정 → 불합격 시 클리닉 대상 자동 등록
    ↓
재시험 허용된 경우 → 학생 재응시 가능
```

### 커뮤니티 관리
```
학생 질문 → 교사 답변 → 학생 확인 (알림톡 자동 발송)
공지 작성 → 학생 앱에 즉시 반영
자료 업로드 → 학생 다운로드 가능
```
